# Oline-Object-Detection
 在线目标识别系统  
 该项目是基于tensorflow+django的框架完成  
 识别部分：模型主要是参考tensorflow的Object detection提供的SSD+MobileNet
 网页部分：是采用django部分完成  
 项目思路：前端页面提供图片上传，上传以后调用目标识别算法，算法给出预测输入，最后将结果呈现在前端页面上
